Edit view file online in the [admin](https://pushword.piedweb.com/extension/admin).

## Install

```shell
composer require pushword/template-editor
```

Add Routes

```yaml
template_editor:
  resource: '@PushwordTemplateEditorBundle/TemplateEditorRoutes.yaml'
```
